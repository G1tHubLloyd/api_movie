GET /movies

Returns a list of all movies in the database.

  • Method: GET
  • URL: /movies
  • Query Parameters: None
  • Request Body: None
  • Response Format: JSON array of movie objects

GET /movies/:id

Returns details of a specific movie by its ID.

  • Method: GET
  • URL: /movies/:id
  • Query Parameters: None
  • Request Body: None
  • Response Format: JSON object of the movie

POST /movies

Adds a new movie to the database.

  • Method: POST
  • URL: /movies
  • Query Parameters: None
  • Request Body Format: JSON
  • Example: 
    {
  "title": "Inception",
  "director": "Christopher Nolan",
  "year": 2010
}
  • Response Format: Confirmation message or created movie object

PUT /movies/:id

Updates an existing movie by its ID.

  • Method: PUT
  • URL: /movies/:id
  • Query Parameters: None
  • Request Body Format: JSON
  • Example: 
    {
  "title": "Inception (Updated)",
  "year": 2011
}
  • Response Format: Confirmation message or updated movie object

DELETE /movies/:id

Deletes a movie by its ID.

  • Method: DELETE
  • URL: /movies/:id
  • Query Parameters: None
  • Request Body: None
  • Response Format: Confirmation message

POST /users

  • Method: POST
  • URL: /users
  • Query Parameters: None
  • Request Body Format: JSON

    {
  "username": "janedoe",

Updates user information by ID.

  • Method: PUT

  • Query Parameters: None
