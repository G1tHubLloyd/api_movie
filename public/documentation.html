Movie API Endpoint Documentation

Request URL Response
GET /movies Returns all movies
GET /movies/:title Returns data about a single movie
GET /genres/:name Returns genre description
GET /directors/:name Returns director bio and birth/death years
POST /users Registers a new user
PUT /users/:username Updates user info
POST /users/:username/favorites/:movieTitle Adds movie to user's favorites
DELETE /users/:username/favorites/:movieTitle Removes movie from user's favorites
DELETE /users/:username Deregisters user
